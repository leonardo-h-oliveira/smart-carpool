# Smart Carpool

## Project Overview

Smart Carpool is a ride-sharing platform designed to connect drivers and passengers in a simple, organized, and secure way.

This project is being developed as a personal portfolio to apply software engineering principles and improve skills in Git, GitHub, Python, FastAPI, PostgreSQL, and web development.

The project is inspired by an academic prototype previously developed as part of an undergraduate capstone project and is now being redesigned from scratch using modern technologies and best development practices.

## Version 1.0 Features

### User Authentication
- User registration
- User login
- Password encryption

### Ride Management
- Create a ride
- Edit a ride
- Cancel a ride
- List available rides

### Ride Requests
- Request a seat
- Accept a request
- Reject a request
- Cancel a request

### User Profile
- View profile
- View created rides
- View requested rides
